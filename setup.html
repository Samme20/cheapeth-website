How To Setup

  1. Create a 600GB storage optimized digital ocean droplet (32 GB ram, 4 CPU)
  2. Create a user:
    adduser goku
    sudo visudo
    # goku ALL=(ALL) NOPASSWD:ALL
  3. su goku
    cd ~/
    mkdir ~/.ssh
    cat > ~/.ssh/authorized_keys
  4. wget https://raw.githubusercontent.com/geohot/configuration/master/.tmux.conf
    git clone https://github.com/geohot/go-ethereum.git -b cheapeth
    cd go-ethereum
    sudo apt install make golang-go
    make geth
  5. ./build/bin/geth --syncmode fast --cache 26000 --rpc --rpcaddr 127.0.0.1 --rpcapi eth,web3,debug,net --rpcvhosts=*
  • For miner (in new window)
    geth attach
    > personal.newAccount()
    > miner.start()
  • For RPC server, setup nginx
    sudo apt-get install nginx
    https://www.nginx.com/resources/wiki/start/topics/recipes/geth/
